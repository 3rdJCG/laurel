## ADDED Requirements

### Requirement: 既存タスクのサブタスクを作成できる
プロジェクト画面のタスクアイテムからサブタスク追加操作を実行できる SHALL。作成されたサブタスクは親タスクの `id` を `parentId` に持つ。

#### Scenario: サブタスク追加ボタンを押下してフォームが表示される
- **WHEN** ユーザーがタスクアイテムのサブタスク追加ボタンを押下する
- **THEN** そのタスクの直下にサブタスクタイトル入力フォームが表示される

#### Scenario: サブタスクが作成されて一覧に追加される
- **WHEN** ユーザーがサブタスクのタイトルを入力して追加を確定する
- **THEN** `parentId` に親タスクの `id` が設定されたサブタスクが作成され、親タスクの直下にインデントして表示される

#### Scenario: サブタスク作成時に親の genre・tags が初期値になる
- **WHEN** ユーザーがサブタスクを作成する
- **THEN** 新しいサブタスクの `genre` と `tags` の初期値に親タスクの値が設定される

### Requirement: サブタスク一覧は親タスクの直下にインデントして表示される
プロジェクト画面でサブタスクは親タスクの直下にネストして表示される SHALL。

#### Scenario: サブタスクがインデントして表示される
- **WHEN** 親タスクにサブタスクが存在する
- **THEN** サブタスクが親タスクの直下にインデント（階層の深さに応じた余白）付きで表示される

### Requirement: 5層目のタスクにはサブタスク追加ボタンが表示されない
階層の深さが5層目のタスクはサブタスクを作成できない SHALL。UIレベルでサブタスク追加ボタンを非表示にすることでブロックする。

#### Scenario: 5層目のタスクにはサブタスク追加ボタンがない
- **WHEN** 第5層のタスクアイテムがプロジェクト画面に表示される
- **THEN** そのタスクアイテムにサブタスク追加ボタンは表示されない
