## ADDED Requirements

### Requirement: タスクのステータスをインラインで変更できる
プロジェクト画面のタスクアイテムに表示されるステータスバッジをクリックすると、ステータス選択UIが展開され、`todo` / `in-progress` / `in-review` / `done` のいずれかに変更できる SHALL。

#### Scenario: ステータスバッジをクリックして選択UIが表示される
- **WHEN** ユーザーがタスクアイテムのステータスバッジをクリックする
- **THEN** `todo` / `in-progress` / `in-review` / `done` の選択肢が表示される

#### Scenario: ステータスを変更すると即時反映される
- **WHEN** ユーザーがステータス選択UIから新しいステータスを選択する
- **THEN** タスクの `status` が更新され、バッジ表示が新しいステータスを反映する

#### Scenario: 任意のステータスに直接変更できる
- **WHEN** ユーザーが現在のステータスとは異なる任意のステータス（例: `done` から `todo`）を選択する
- **THEN** `status` が選択した値に更新される（順次サイクルに限定されない）

### Requirement: ステータスは視覚的に識別できる
各ステータス値は固有の視覚スタイル（色・ラベル）で表示される SHALL。

#### Scenario: ステータスバッジの視覚的識別
- **WHEN** タスクアイテムが一覧に表示される
- **THEN** ステータスバッジの色またはラベルによってステータス値を一見して識別できる
