## MODIFIED Requirements

### Requirement: プロジェクトのタスク一覧表示
プロジェクト画面は、そのプロジェクトに属するタスクを階層構造を考慮して一覧表示する SHALL。ルートタスク（`parentId` が null のタスク）のみを初期表示し、子タスクは折りたたみ状態とする。タスクはプロジェクト内の `order` フィールド順に表示される。

#### Scenario: タスクが存在する場合
- **WHEN** ユーザーがプロジェクト画面を開く
- **THEN** そのプロジェクトに属するルートタスクが `order` 順に一覧表示される

#### Scenario: タスクが存在しない場合
- **WHEN** ユーザーがプロジェクト画面を開き、タスクが1件もない
- **THEN** タスクが空である旨のメッセージが表示される
