## ADDED Requirements

### Requirement: プロジェクト名をインライン編集できる
Home 画面のプロジェクトカードから編集操作を起動し、プロジェクト名を変更できる SHALL。

#### Scenario: 編集ボタン押下で名前が入力フィールドになる
- **WHEN** ユーザーがプロジェクトカードの編集ボタン（鉛筆アイコン）を押下する
- **THEN** プロジェクト名が編集可能な入力フィールドとして表示される

#### Scenario: 新しい名前を確定すると更新される
- **WHEN** ユーザーが入力フィールドに新しい名前を入力して確定操作（Enterキーまたは保存ボタン）を行う
- **THEN** プロジェクトの `name` が更新され、カードに新しい名前が表示される

#### Scenario: キャンセルすると変更前の名前に戻る
- **WHEN** ユーザーがキャンセル操作（Escapeキーまたはキャンセルボタン）を行う
- **THEN** プロジェクト名は変更前の値のままカード表示に戻る

#### Scenario: 空の名前では確定できない
- **WHEN** ユーザーがプロジェクト名を空にして確定操作を行う
- **THEN** 変更が拒否され、編集モードが維持される

### Requirement: 編集中は他のプロジェクトカードへのナビゲーションが無効になる
プロジェクト名を編集中は、意図せず画面遷移が発生しない SHALL。

#### Scenario: 編集中にカードのクリックエリアが無効化される
- **WHEN** ユーザーがプロジェクト名を編集中にカードのナビゲーション領域をクリックする
- **THEN** プロジェクト画面への遷移は発生せず、編集モードが維持される
