## ADDED Requirements

### Requirement: プロジェクト画面でジャンルによるフィルタリングができる
プロジェクト画面のタスク一覧をジャンルで絞り込めるフィルター UI を提供する SHALL。

#### Scenario: ジャンルを選択してタスクを絞り込む
- **WHEN** ユーザーがフィルターバーのジャンルドロップダウンからジャンルを選択する
- **THEN** 選択したジャンルに一致するタスクのみ表示され、他は非表示になる

#### Scenario: ジャンルフィルターをリセットする
- **WHEN** ユーザーがジャンルフィルターを「すべて」に戻す
- **THEN** ジャンルによる絞り込みが解除され、すべてのタスクが表示される

#### Scenario: フィルター後にジャンルに一致するタスクがない場合
- **WHEN** 選択したジャンルを持つタスクが存在しない状態でフィルタリングする
- **THEN** タスクが表示されず、「該当するタスクがありません」のメッセージが表示される

### Requirement: プロジェクト画面でタグによるフィルタリングができる
プロジェクト画面のタスク一覧をタグで絞り込めるフィルター UI を提供する SHALL。複数タグを指定した場合はAND条件で絞り込む。

#### Scenario: 1つのタグを選択してタスクを絞り込む
- **WHEN** ユーザーがフィルターバーのタグセレクターから1つのタグを選択する
- **THEN** 選択したタグを持つタスクのみ表示される

#### Scenario: 複数タグを選択してAND条件で絞り込む
- **WHEN** ユーザーが複数のタグを選択する
- **THEN** 選択したすべてのタグを持つタスクのみ表示される

#### Scenario: タグフィルターをリセットする
- **WHEN** ユーザーがタグフィルターを解除する
- **THEN** タグによる絞り込みが解除され、すべてのタスクが表示される

### Requirement: ジャンルとタグのフィルターを組み合わせられる
ジャンルフィルターとタグフィルターを同時に適用できる SHALL。組み合わせ時はAND条件で絞り込む。

#### Scenario: ジャンルとタグを同時に指定してフィルタリングする
- **WHEN** ユーザーがジャンルとタグの両方を指定してフィルタリングする
- **THEN** 指定したジャンル かつ 指定したすべてのタグを持つタスクのみ表示される

### Requirement: フィルター状態は画面遷移でリセットされる
フィルター状態はコンポーネントのローカル state として保持し、プロジェクト画面から離脱すると初期化される SHALL。

#### Scenario: 別プロジェクトに移動するとフィルターがリセットされる
- **WHEN** ユーザーがフィルターを適用した状態で別のプロジェクト画面に移動する
- **THEN** フィルターが初期状態（すべて表示）にリセットされる
