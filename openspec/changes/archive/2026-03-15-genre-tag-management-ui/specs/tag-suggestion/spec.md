## ADDED Requirements

### Requirement: タグ入力フィールドで過去使用タグのサジェストを提供する
タスク作成・編集フォームのタグ入力フィールドに、プロジェクト内で過去に使用されたタグの候補を表示する SHALL。

#### Scenario: タグ入力フィールドを操作すると候補が表示される
- **WHEN** ユーザーがタスクフォームのタグ入力フィールドをフォーカスまたは文字を入力する
- **THEN** 入力中の文字列にマッチするプロジェクト内の既存タグが候補一覧として表示される

#### Scenario: 候補から選択してタグを追加できる
- **WHEN** ユーザーがサジェスト候補のタグをクリックまたはキーボードで選択する
- **THEN** 選択したタグがタスクの tags フィールドに追加される

#### Scenario: 候補にないタグを自由入力で追加できる
- **WHEN** ユーザーがサジェスト候補にない文字列を入力して確定する
- **THEN** 入力した文字列がそのまま tags フィールドに追加される

#### Scenario: 既に追加済みのタグは候補から除外される
- **WHEN** ユーザーがタグ入力フィールドで候補を参照する
- **THEN** 既に同タスクに設定済みのタグは候補一覧に表示されない

### Requirement: タグ候補はプロジェクト内の全タスクから収集される
サジェスト候補はプロジェクト内の全タスクの `tags` フィールドを集計したものを使用する SHALL。プロジェクトをまたいだ候補収集は行わない。

#### Scenario: 過去のタスクで使用したタグが候補に表示される
- **WHEN** プロジェクト内の既存タスクに設定されているタグと同じ文字列を入力する
- **THEN** そのタグが候補として表示される

#### Scenario: 別プロジェクトのタグは候補に表示されない
- **WHEN** ユーザーがプロジェクトAのタスクにタグを追加する際に候補を参照する
- **THEN** プロジェクトBのタスクにのみ存在するタグは候補に表示されない
