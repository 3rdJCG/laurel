## ADDED Requirements

### Requirement: Home 画面からプロジェクトをインラインフォームで作成できる
Home 画面のプロジェクト一覧末尾に「＋ プロジェクトを追加」ボタンが表示され、押下するとインラインのプロジェクト名入力フォームが展開される SHALL。

#### Scenario: ボタン押下でフォームが展開される
- **WHEN** ユーザーが「＋ プロジェクトを追加」ボタンを押下する
- **THEN** プロジェクト一覧の末尾にプロジェクト名入力フォームが展開される

#### Scenario: プロジェクト名を入力して Enter キーで作成できる
- **WHEN** ユーザーがフォームにプロジェクト名を入力して Enter キーを押す
- **THEN** 新しいプロジェクトが作成され、一覧の末尾に表示される。フォームは閉じられる

#### Scenario: 確定ボタンでプロジェクトを作成できる
- **WHEN** ユーザーがフォームにプロジェクト名を入力して確定ボタンを押す
- **THEN** 新しいプロジェクトが作成され、一覧の末尾に表示される。フォームは閉じられる

#### Scenario: Escape キーでフォームがキャンセルされる
- **WHEN** ユーザーがフォーム表示中に Escape キーを押す
- **THEN** フォームが閉じられ、プロジェクトは作成されない

#### Scenario: プロジェクト名が空のまま確定しても作成されない
- **WHEN** ユーザーがプロジェクト名を入力せずに確定操作（Enter キーまたは確定ボタン）を行う
- **THEN** プロジェクトは作成されず、フォームはそのまま表示され続ける

### Requirement: 作成されたプロジェクトは永続化される
プロジェクトを作成した後、`data:save-project` IPC を通じてデータが保存される SHALL。

#### Scenario: プロジェクト作成後にデータが保存される
- **WHEN** ユーザーがプロジェクトの作成を確定する
- **THEN** プロジェクトデータが `data:save-project` で保存され、アプリを再起動しても作成したプロジェクトが表示される
