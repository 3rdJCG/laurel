## ADDED Requirements

### Requirement: タスクを削除できる
タスクアイテムから削除操作を実行できる SHALL。誤削除を防ぐために確認ステップが必要である。

#### Scenario: 削除ボタンを押下すると確認UIが表示される
- **WHEN** ユーザーがタスクアイテムの削除ボタンを押下する
- **THEN** タスクアイテム内に確認UI（「本当に削除しますか？」「削除」「キャンセル」ボタン）が表示される

#### Scenario: 確認後にタスクが削除される
- **WHEN** 確認UIでユーザーが「削除」を選択する
- **THEN** タスクが削除され、一覧から消える

#### Scenario: キャンセルすると削除されない
- **WHEN** 確認UIでユーザーが「キャンセル」を選択する
- **THEN** タスクは削除されず、確認UIが閉じて通常表示に戻る

### Requirement: 親タスクを削除すると子タスクも連鎖削除される
サブタスクを持つ親タスクを削除した場合、すべての子孫タスクも削除される SHALL。

#### Scenario: 子タスクを持つ親タスクの削除確認
- **WHEN** サブタスクを持つ親タスクの削除ボタンを押下する
- **THEN** 確認UIに子タスクも削除される旨の警告が表示される

#### Scenario: 親タスク削除時に子孫タスクが連鎖削除される
- **WHEN** ユーザーが親タスクの削除を確認する
- **THEN** 親タスクおよびそのすべての子孫タスクが削除され、一覧から消える
