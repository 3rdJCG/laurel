## ADDED Requirements

### Requirement: タスクのタイトルをインライン編集できる
タスクアイテムの編集操作（編集ボタン押下またはダブルクリック）により、タイトルが編集可能な入力フィールドに切り替わる SHALL。

#### Scenario: 編集モードに入る
- **WHEN** ユーザーがタスクアイテムの編集ボタンを押下する
- **THEN** タスクのタイトルが編集可能な入力フィールドとして表示される

#### Scenario: タイトルを変更して確定する
- **WHEN** ユーザーが入力フィールドを変更し、確定操作（Enterキーまたは保存ボタン）を行う
- **THEN** タスクの `title` が入力した値に更新され、表示モードに戻る

#### Scenario: 編集をキャンセルする
- **WHEN** ユーザーがキャンセル操作（Escapeキーまたはキャンセルボタン）を行う
- **THEN** タイトルは変更前の値に戻り、表示モードに戻る

#### Scenario: 空のタイトルでは確定できない
- **WHEN** ユーザーがタイトルを空にして確定操作を行う
- **THEN** 変更が拒否され、編集モードが維持される

### Requirement: タスクの genre をインライン編集できる
編集モード中に genre フィールドを変更できる SHALL。未設定（`null`）への変更も可能である。

#### Scenario: genre を設定する
- **WHEN** 編集モードでユーザーが genre 入力フィールドに文字列を入力して確定する
- **THEN** タスクの `genre` が入力した文字列に更新される

#### Scenario: genre を削除して null にする
- **WHEN** 編集モードでユーザーが genre 入力フィールドを空にして確定する
- **THEN** タスクの `genre` が `null` に更新される

### Requirement: タスクの tags をインライン編集できる
編集モード中に tags フィールドを変更できる SHALL。タグは複数設定でき、個別に追加・削除が可能である。

#### Scenario: タグを追加する
- **WHEN** 編集モードでユーザーがタグ入力フィールドに文字列を入力してタグを追加する
- **THEN** 入力した文字列が `tags` 配列に追加される

#### Scenario: タグを削除する
- **WHEN** 編集モードでユーザーが既存タグの削除操作を行う
- **THEN** 対象タグが `tags` 配列から除去される

### Requirement: 同時に複数のタスクを編集モードにできない
任意の時点で編集モードになれるタスクは1件のみである SHALL。

#### Scenario: 別のタスクの編集を開始すると現在の編集がキャンセルされる
- **WHEN** タスクAが編集モード中にユーザーがタスクBの編集ボタンを押下する
- **THEN** タスクAの編集がキャンセルされ、タスクBが編集モードになる
