## ADDED Requirements

### Requirement: MIT ライセンスファイルの提供
リポジトリのルートに `LICENSE` ファイルが存在し、MIT ライセンスの全文と著作権表示を含む SHALL。

#### Scenario: LICENSE ファイルが存在する
- **WHEN** ユーザーがリポジトリのルートディレクトリを確認する
- **THEN** `LICENSE` ファイルが存在し、MIT ライセンスの全文が記載されている

#### Scenario: 商用利用が許可されている
- **WHEN** ユーザーが LICENSE の内容を確認する
- **THEN** 商用利用・改変・再配布・私的利用がすべて許可されていることが明記されている

### Requirement: README によるプロジェクト説明
リポジトリのルートに `README.md` が存在し、以下を含む SHALL:
- プロジェクトの概要と目的
- 主要機能の説明
- インストール・起動手順
- 開発環境のセットアップ手順
- ライセンス情報への参照

#### Scenario: README が存在する
- **WHEN** ユーザーがリポジトリのルートディレクトリを確認する
- **THEN** `README.md` ファイルが存在する

#### Scenario: 開発手順が記載されている
- **WHEN** ユーザーが README を参照する
- **THEN** `npm install` と `npm run dev` で開発環境を起動できる手順が記載されている

### Requirement: プライバシーポリシー文書の提供
`docs/PRIVACY.md` にプライバシーポリシーが存在し、以下を含む SHALL:
- 収集する情報（なし）の明記
- 外部へのデータ送信がないことの明記
- データの保存場所（ローカルのみ）の明記
- 適用ライセンスへの参照

#### Scenario: プライバシーポリシーが存在する
- **WHEN** ユーザーが `docs/PRIVACY.md` を確認する
- **THEN** ファイルが存在し、プライバシーに関する方針が記載されている

#### Scenario: 外部送信なしの明記
- **WHEN** ユーザーがプライバシーポリシーを読む
- **THEN** ユーザーデータが外部サーバーに送信されないことが明確に記載されている
