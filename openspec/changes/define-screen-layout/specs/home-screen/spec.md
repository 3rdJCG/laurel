## ADDED Requirements

### Requirement: プロジェクト一覧の表示
Home 画面は作成済みのプロジェクトを一覧表示する SHALL。

#### Scenario: プロジェクトが存在する場合
- **WHEN** ユーザーが Home 画面を開く
- **THEN** 作成済みのすべてのプロジェクトが一覧で表示される

#### Scenario: プロジェクトが存在しない場合
- **WHEN** ユーザーが Home 画面を開き、プロジェクトが1件もない
- **THEN** プロジェクトが空である旨のメッセージが表示される

### Requirement: プロジェクトの追加
Home 画面からプロジェクトを新規作成できる SHALL。プロジェクトは複数作成可能である。

#### Scenario: プロジェクトを追加する
- **WHEN** ユーザーがプロジェクト名を入力してプロジェクト追加を実行する
- **THEN** 新しいプロジェクトが作成され、一覧に追加される

#### Scenario: 複数のプロジェクトを作成できる
- **WHEN** ユーザーが複数回プロジェクト追加を実行する
- **THEN** すべてのプロジェクトが一覧に表示される

### Requirement: タスクの新規追加とプロジェクトへの分類
Home 画面からタスクを新規作成し、既存のプロジェクトに分類できる SHALL。

#### Scenario: タスクをプロジェクトに分類して追加する
- **WHEN** ユーザーがタスクタイトルを入力し、分類先プロジェクトを選択してタスク追加を実行する
- **THEN** タスクが作成され、選択したプロジェクトに紐付けられる

### Requirement: プロジェクト画面へのナビゲーション
Home 画面からプロジェクトを選択して、そのプロジェクトの画面に遷移できる SHALL。

#### Scenario: プロジェクトを選択して遷移する
- **WHEN** ユーザーが一覧のプロジェクトを選択する
- **THEN** そのプロジェクトのタスク一覧画面に遷移する
