# プライバシーポリシー

Laurel はローカルファーストのタスク管理アプリです。このドキュメントでは、Laurel が保存するデータとネットワーク接続について説明します。

## 収集する情報

Laurel は分析・広告・その他の第三者目的で個人情報を収集しません。

以下のデータは**お使いのコンピューター上にのみ**保存されます：

- **タスクデータ** — 作成したプロジェクト、タスク、イシュー、コメント。設定で指定したフォルダに JSON ファイルとして保存されます（デフォルト: `~/Documents/laurel/`）
- **アプリ設定** — データフォルダのパス、ジャンル一覧、アップデートチャンネル設定、および設定画面で任意入力した表示名とメールアドレス。システムの userData ディレクトリに保存されます。

表示名とメールアドレスは、作成したイシューやコメントの著者情報としてのみ使用され、外部に送信されることはありません。

## ネットワーク接続

Laurel が行う外部通信は **アップデート確認のみ** です。[GitHub Releases API](https://docs.github.com/en/rest/releases) を使用します。

- **タイミング**: アプリ起動時および起動中4時間ごと
- **送信される情報**: アプリのバージョン、OS、プラットフォーム — GitHub API が新しいリリースの有無を判断するために必要な標準的な情報
- **送信されない情報**: タスク、設定、表示名、メールアドレス、その他の個人データ

分析・クラッシュレポート・テレメトリ等は一切収集していません。

## データの保存場所

| データ | 保存場所 |
|--------|----------|
| タスクデータ（プロジェクト、タスク、イシュー、コメント） | 設定で指定したフォルダ（デフォルト: `~/Documents/laurel/`） |
| アプリ設定（ジャンル、データフォルダ、アップデートチャンネル、表示名、メールアドレス） | `{userData}/laurel/settings.json` |

`{userData}` は Electron の userData ディレクトリです：
- **Windows**: `%APPDATA%\laurel`
- **macOS**: `~/Library/Application Support/laurel`
- **Linux**: `~/.config/laurel`

## データの削除

すべてのデータを削除するには：

1. アプリをアンインストールする
2. データフォルダを削除する（設定画面 → データ で確認できます）
3. 上記の `{userData}/laurel/` ディレクトリを削除する
