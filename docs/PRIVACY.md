# プライバシーポリシー

Laurel はローカルで完結するタスク管理アプリです。

## 収集する情報

Laurel はいかなる個人情報も収集しません。

## 外部への情報送信

Laurel はインターネットへの通信を一切行いません。すべてのデータはお使いのコンピューター上にのみ保存されます。

## データの保存場所

タスクデータは、設定画面で指定したローカルフォルダに JSON ファイルとして保存されます。アプリ設定（保存先パス・ジャンル一覧）はシステムの userData ディレクトリに保存されます。

## データの削除

アプリをアンインストールし、データフォルダおよび userData ディレクトリを削除することで、すべてのデータを削除できます。
